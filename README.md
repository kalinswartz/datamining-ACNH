# datamining-ACNH
CSCE 676: Data Mining and Analysis Project on Animal Crossing: New Horizons Catalog of Villagers

## Project Overview
This project explores the Animal Crossing: New Horizons – Nook Plaza dataset from Kaggle (with a focus on the Villagers.csv data) using exploratory data analysis (EDA) and data mining techniques.

The primary focus is on:
- Villager demographic attributes (species, personality, gender, hobby)
- Birthday patterns
- Furniture “basket” analysis
- Statistical relationships between categorical variables
- Frequent itemset and co-occurrence analysis
  
The goal is to apply course concepts such as:
- Chi-square analysis
- Frequent itemset mining
- Association rules
- Co-occurrence analysis
- Clustering
- Possible Graph & community detection
- Decision trees/logistic regression

## Dataset
Dataset source: https://www.kaggle.com/datasets/jessicali9530/animal-crossing-new-horizons-nookplaza-dataset

Files used include:
- villagers.csv
- housewares.csv
- miscellaneous.csv
- wall-mounted.csv
